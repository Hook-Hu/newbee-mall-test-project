# 测试文档目录

本目录用于沉淀 `newbee-mall-plus` 项目的测试资料，覆盖项目理解、测试计划、测试用例、接口测试、自动化测试、性能测试、缺陷记录、测试报告和面试复盘。

## 目录说明

| 目录 | 说明 |
| --- | --- |
| `01_project_overview` | 项目概览、业务模块、测试范围说明 |
| `02_test_plan` | 测试计划、资源安排、风险和准入准出标准 |
| `03_test_cases` | 功能测试用例、场景用例、回归用例 |
| `04_api_testing` | 接口测试范围、接口清单、断言和数据准备 |
| `05_automation_testing` | 自动化测试方案、脚本结构、执行说明 |
| `06_performance_testing` | 性能测试目标、场景设计、指标和瓶颈分析 |
| `07_bug_reports` | 缺陷模板、缺陷跟踪和典型问题记录 |
| `08_test_reports` | 阶段测试报告、回归报告、上线测试结论 |
| `09_interview_review` | 项目面试复盘、测试亮点、问题回答准备 |
