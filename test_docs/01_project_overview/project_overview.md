# 项目概览

## 项目名称

`newbee-mall-plus` 商城测试项目。

## 项目定位

本项目是一个基于 `Spring Boot` 的商城系统，适合作为功能测试、接口测试、自动化测试、性能测试和测试面试项目复盘材料。

## 核心业务模块

| 模块 | 测试关注点 |
| --- | --- |
| 首页 | 轮播图、商品分类、推荐商品、页面资源加载 |
| 商品 | 商品列表、商品详情、图片展示、库存和价格展示 |
| 搜索 | 关键字搜索、分类筛选、分页、排序 |
| 购物车 | 加入购物车、数量修改、删除、勾选和金额计算 |
| 订单 | 下单、订单详情、订单状态流转、取消订单 |
| 用户 | 注册、登录、退出、个人信息、收货地址 |
| 后台管理 | 管理员登录、商品管理、分类管理、订单管理 |
| 秒杀 | 秒杀商品展示、库存扣减、并发下单控制 |

## 测试范围

- 前台商城核心购买流程。
- 后台商品、订单、分类等基础管理流程。
- 图片资源、本地上传目录和静态资源访问。
- 接口参数校验、业务状态校验和异常场景。
- 秒杀、下单、购物车等高风险业务链路。

## 不在本轮测试范围

- 第三方支付真实交易。
- 线上短信、邮件等外部服务。
- 多端兼容性深度测试。
- 生产级安全渗透测试。
