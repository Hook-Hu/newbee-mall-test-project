# newbee-mall-plus 接口测试范围

## 一、测试目标

本轮接口测试围绕 newbee-mall-plus 商城系统核心业务链路展开，重点验证用户登录、商品浏览、购物车、订单、优惠券、秒杀等接口的请求参数、响应结果、登录态校验和异常处理能力。

## 二、本轮重点测试接口

| 优先级 | 模块 | 接口名称 | 请求方法 | 接口地址 | 测试重点 |
|---|---|---|---|---|---|
| P0 | 公共接口 | 商城验证码 | GET | `/common/mall/kaptcha` | 验证码是否正常返回，Session 是否建立 |
| P0 | 用户模块 | 用户登录 | POST | `/login` | 正常登录、参数为空、验证码错误、密码错误 |
| P0 | 用户模块 | 用户注册 | POST | `/register` | 正常注册、重复账号、验证码错误、参数为空 |
| P0 | 商品模块 | 商品搜索 | GET | `/search` | 关键词搜索、分类筛选、分页、排序 |
| P0 | 商品模块 | 商品详情 | GET | `/goods/detail/{goodsId}` | 正常商品、无效商品 ID、下架商品 |
| P0 | 购物车模块 | 查看购物车 | GET | `/shop-cart` | 登录态校验、购物车页面访问 |
| P0 | 购物车模块 | 添加购物车 | POST | `/shop-cart` | 正常添加、商品不存在、数量异常、未登录 |
| P0 | 购物车模块 | 修改购物车 | PUT | `/shop-cart` | 修改数量、数量边界、购物车项不存在 |
| P0 | 购物车模块 | 删除购物车 | DELETE | `/shop-cart/{newBeeMallShoppingCartItemId}` | 正常删除、无效购物车项、未登录 |
| P0 | 订单模块 | 创建普通订单 | GET | `/saveOrder` | 正常下单、未登录、购物车为空、优惠券参数 |
| P0 | 订单模块 | 取消订单 | PUT | `/orders/{orderNo}/cancel` | 正常取消、订单不存在、非本人订单 |
| P0 | 优惠券模块 | 领取优惠券 | POST | `/coupon/{couponId}` | 正常领取、重复领取、优惠券不存在、未登录 |
| P0 | 秒杀模块 | 秒杀商品列表 | GET | `/seckill/list` | 秒杀商品列表是否正常返回 |
| P0 | 秒杀模块 | 秒杀地址暴露 | POST | `/seckill/{seckillId}/exposer` | 活动时间校验、库存校验、秒杀地址生成 |
| P0 | 秒杀模块 | 秒杀执行 | POST | `/seckillExecution/{seckillId}/{userId}/{md5}` | 正常秒杀、重复秒杀、库存不足、md5 错误 |

## 三、暂不重点测试接口

以下接口本轮暂不作为重点：

- 后台轮播图管理接口；
- 首页配置管理接口；
- 文件上传接口；
- 错误处理接口；
- 后台个人资料修改接口。

原因：本轮测试优先覆盖商城核心交易链路，后台配置类接口后续补充测试。
